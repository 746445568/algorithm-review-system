# OJ 复盘系统桌面端本地服务

本目录包含 Windows 桌面版重构所用的 Go 本地服务。

## 当前范围

- 仅限本地的 HTTP 服务，监听 `127.0.0.1`
- 基于 SQLite 存储用户档案、平台账号、题目、提交记录、同步任务、复习快照、分析任务和应用设置
- 数据库迁移前自动备份
- 持久化的进程内任务队列框架，支持重启恢复
- 加密的本地 AI 设置存储
- 面向桌面端的 REST 接口：健康检查、账号管理、同步任务创建、AI 设置、诊断导出、分析任务创建
